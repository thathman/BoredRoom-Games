# Code of conduct

BoredRoom Games should be a respectful and welcoming project space.

## Expected behavior

- Be respectful and constructive.
- Give clear feedback without personal attacks.
- Keep discussions focused on the project.
- Credit contributors where appropriate.

## Unacceptable behavior

- Harassment, threats, or abuse.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Spamming issues, discussions, or pull requests.

## Enforcement

Maintainers may edit, hide, close, or remove comments, issues, pull requests, or contributions that do not follow this code of conduct.
